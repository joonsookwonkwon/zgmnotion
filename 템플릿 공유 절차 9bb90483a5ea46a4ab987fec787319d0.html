템플릿 공유 절차

0. 준비할 파일

2D 크리에이터 분들의 작업을 위해서 필요한 파일은 2가지 입니다:

  • .zepeto 파일
  • .psd 텍스쳐 파일

이때 텍스쳐 파일에는 UV 맵 레이어와 영역 표시 레이어가 둘 다 포함되어야 합니다.

1. 드라이브 업로드

3D 템플릿 공유 드라이브 (링크) 에서 경로에 맞게 준비한 파일을 업로드 합니다.

본인 폴더에서 카테고리에 맞게 올려야 하며, 아이템 개별 폴더명은

템플릿유형_3D고유코드_카테고리_넘버링 형식으로 수정합니다.

예) P_3D100ZD_DR_03: “ZDE의 DR 카테고리에서 세번째로 업로드된 P_템플릿"

O_3D111XO_TOP_001: “JM의 TOP 카테고리에서 처음으로 업로드된 O_템플릿”


해당 카테고리에 처음으로 공유하는 경우 넘버링은 01이며,
이후 동일한 카테고리에 계속 추가할 때 마다 파일명에서 넘버링만 달라집니다. (02, 03, 04…)

2. 시트 입력

3D 템플릿 공유 시트 (링크)에서 아이템 정보를 기입해 템플릿을 등록합니다.

본인 시트에서 카테고리에 맞게 올리며, 상단부터 차례대로 한 줄씩 기입합니다.

다음을 참고해서 칸을 채워주세요:

*특이사항 같은 경우 글로벌 크리에이터들을 위해 영어로도 짧게 적어주시는 것을 권장합니다.

3. 디스코드 공지

신규 템플릿이 나왔다는 소식을 2D 크리에이터분들이 아실 수 있도록

디스코드 신규 템플릿 채널 new-template 채널 (링크)에 공지해 홍보합니다.

필요한 내용은 시트에 기입한 것과 비슷합니다:

NEW TEMPLATE
Creator:
본인 크리에이터 이름
Naming :
템플릿 파일명
Category :
DR/BTM/SH/HW/GL...
ZEM :
정상가 (+세일 최저가)
Share:
구글 드라이브 링크
