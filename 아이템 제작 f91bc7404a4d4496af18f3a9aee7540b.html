2. 기존 디자인 확인 (1차)

🌈디자인 규칙에 의해 이미 원작자나 매니지먼트 내에 다른 크리에이터가
출시한 텍스쳐와 차별화된 텍스쳐를 입혀야 한다.

O_템플릿은 디스코드 promo-feed-share 채널 (링크) 에서 템플릿 파일명으로 검색해
본인이 피해야 하는 기존 디자인을 미리 참고해야 하며
P_템플릿 같은 경우 다른 2D 크리에이터 뿐만 아니라
3D 템플릿 원작자의 본계, 부계까지 확인 해야 한다.

계정 아이디는 👥3D 크리에이터 소개 혹은 원작자의 디스코드 프로필에서 확인

3. 파일 다운로드

3D 템플릿 공유 드라이브 (링크)에서
.zepeto 파일과 텍스쳐 .psd 파일을 둘 다 다운로드 받는다.

.zepeto 파일

  • 다운로드 받았다면 파일명 끝을 .zepeto 에서 .zip으로 수정해 압축 파일로 만든다
  • 방금 만든 압축 파일을 푼다
  • 푼 파일은 유니티 폴더에 넣어준다

.psd 파일

  • 다음 텍스쳐 작업에서 설명
