ZGM 매거진 (룩북)

매거진 소개
  • 2주마다 올라가는 ZGM 전용 매거진으로, ZGM 크리에이터들이 세일하는 신상 아이템을 홍보할 수 있는 정기 콘텐츠 입니다.
  • 편집된 영상은 참여한 크리에이터들의 태그와 함께
    ZDE 제페토 계정과 인스타그램 계정 피드에 올라갑니다.
  • 해당 아이템들을 착용하고 콘텐츠를 만드는 유저들은 추첨으로 젬 충전을 받을 수 있습니다.

참여 방법
1. .Zip 파일
- 하나로 압축해야 하는 3가지 파일: 아이템 png (최대 6개) + 계정 정보 txt + 아바타 png
- .zip 파일명 = 본인 유저명과 동일해야 함
- 주의사항 = .rar 파일로 압축하지 않기 (일부 기기에서 열 수 없는 형태)
1. 아이템 .png
  • 제출 가능한 아이템
    ZGM 템플릿으로 만든 아이템 중에서
    신상 (가장 최근 매거진 업로드 이후 본인 샵에 출시된 아이템들)
  • 아이템 제출 개수
    1~6 (6개가 최대)
  • 아이템 썸네일
    제페토 스튜디오에서 다운로드 받은 .png 파일 (자동 생성, 투명 배경)

2. 아바타 .png
  • 1명의 아바타만 포함한 이미지 1개
    크리에이터 본인이 소유한 캐릭터 중
    하나여야함
    2명 이상의 아바타 X
    1명의 아바타 + 이미지 여러 개 제출 X

  • 아바타 포즈
    제페토 전신 포토부스 아무거나
    (낮은 퀄리티)
    카메라 기능에서 촬영한 전신샷 아무거나 (높은 퀄리티)

  • 파일 포맷
    .png 파일 (투명 배경)
3. 계정 아이디 .txt
  • 필요한 정보 2가지
    본인 유저명 (주로 본계 기준)
    해당 샵 제페토 ID (본인이 제출하는 아이템 대부분이 어떤 계정에 출시했는지 참고)

  • 주의사항
    일반 한글이나 알파벳 아닌 다른 글자들은 편집자가 파일을 열 때 왜곡될 가능성이 큼.
    특수 문자는 가급적 피하기

  • txt 파일명
    본인 유저명과 동일
2. 신청
  • Zip 파일 업로드 하는 채널과 시간
    #upload-file 채널에서,
    금요일 혹은 월요일 봇 리마인더 이후

  • 자동 봇 리마인더 (격주)
    월 10:00) 매거진 신청 오픈
    화 23:59) 매거진 신청 마감
    금 11:30) 매거진 업로드 리마인더

  • 주의사항
    한 매거진 당 받을 수 있는 신청 파일은 올바른 포맷으로 제출한 선착순 10개까지.
    잘못된 포맷으로 제출된 파일은 선착순 안에 들어도 포함될 수 없음.

3. 피드 업로드 날
  • 아이템 세일
    노출된 아이템들은 자율적으로 세일해서 홍보 효과 늘리기

  • 태그 준비
    매거진은 금요일 오후 1~2시쯤에 크리에이터 태그와 함께 ZDE 피드에 업로드 됨.
    해당 아이템 착장 후 태그 설정 열어 놓아서 태그 기능 활용하기

  • 커버 모델 규칙
    매거진 커버엔 센터 메인 모델과 함께 참여한 나머지 크리에이터들이 작게 소개됨.
    메인 모델은 내부 어드민이 결정하며, 한번 메인 모델로 등장한 크리에이터는
    다음 매거진 2회 동안 메인으로 등장할 수 없음.
