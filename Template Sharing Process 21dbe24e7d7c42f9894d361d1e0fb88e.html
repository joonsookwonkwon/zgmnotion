Template Sharing Process

0. Files to prepare

For the 2D Creators to be able to work, 2 files are needed to be shared from the 3D Creator:

  • .zepeto file
  • .psd texture file

Within the texture file, both UV map layer and Section distinction layer must be included.

1. Uploading on Drive

On 3D Template share Drive (Link), upload the prepared files in the proper route.

3D Creators must upload under their own folder > respective category, with the item folder named in the following format:

Template type_3Dindividualcode_Category_Numbering .

Ex) P_3D100ZD_DR_03: “ZDE’s P_Template, 3rd uploaded item under DR category"

O_3D111XO_TOP_001: “JM’s O_Template, first item to be uploaded under TOP category”


The numbering is 01 or 001 if it’s the first item to be uploaded in that category.
For further items added under that category, their names will only differ in numbering
(02, 03, 04…)

2. Writing on Sheet

On 3D Template share Sheet (Link), write down the item details to register the template.

3D Creators must upload under their own sheet > respective category, and fill out starting from the topmost row.
